# Groundup Test API

#Requirement
```
Node :  v14.20.0
NPM : 6.14.17
MongoDB: 6.0.5
```

# How to Install

1. Add Environtment files .env
params: 
```
DATABASE_URL= <<your mongo database url >>
PORT= <<your opened port>>
```

2. Install Depedencies
```
npm install
```

3. Start Server
```
npm start
```
